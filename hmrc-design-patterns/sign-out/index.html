Sign out

This is the GOV.UK header with an extra part to let the user sign out of a service.

When to use

Use this version of the GOV.UK header when the user is signed in to a service, to enable them to sign out.

When not to use

Do not use when the service does not require the user to sign in, or when they are already signed out. Use the default GOV.UK header instead.

How it works

The sign out link should be inside the GOV.UK header, on the right hand side.
