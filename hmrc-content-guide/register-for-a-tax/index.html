    Register once for a tax, duty or levy

    When to use

    Use this pattern when a user needs to register for a tax, duty or levy only once or not very often.

    When not to use

    Do not use this pattern when the user needs to:

    • register a product, item or license
    • only enrol or register for an online service
    • register often for a tax, for example, if the user is acting on behalf of someone else

    How it works

    Detailed guide.

    Optional elements

    You do not need to include a “When to register” sub-heading if you can include this information in “Who should register”.
